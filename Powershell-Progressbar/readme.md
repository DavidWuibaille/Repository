# 📊 Progress Bar Script

## 🔧 What it does
Displays a **PowerShell progress bar** while looping through a list of computers.  
Shows the current index and total count, then closes the bar at the end.

## ⚠️ Notes
- `$OrdinateurClient` must contain a `DISPLAYNAME` column.  
- Run in **PowerShell 5+**.
