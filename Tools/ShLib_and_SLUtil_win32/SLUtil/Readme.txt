Shell Libraries Utility - Native (SLUtil.exe)
==============================================
Uses native APIs to programatically perform operations on a Windows library.

Requires Microsoft Visual C++ 2010 Redistributable Package
x86: http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=a7b7a05e-6de6-4d3a-a423-37bf0912db84
x64: http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=bd512d9e-43c8-4655-81bf-9350143d5867
